# Layer Model

## system

Rules, workflows, schemas, registries, and adapter files.

## intake

Raw capture and machine-landed material such as:

- imported files
- source snapshots
- NotebookLM outputs
- quick captures

## curation

Reorganized working knowledge:

- thematic indexes
- draft summaries
- merged source notes
- working concept maps

## canonical

Durable, high-findability knowledge intended to outlive the current research cycle.

## archive

Frozen, retired, or historically preserved material that should not drift silently.
