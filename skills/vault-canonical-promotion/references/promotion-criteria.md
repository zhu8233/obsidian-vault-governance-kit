# Promotion Criteria

A note is promotion-ready when it satisfies most of these:

- stable enough to survive outside the current research cycle
- understandable without reopening all raw inputs
- linked to a clear topic
- supported by at least one explicit source or upstream object
- more useful as a durable reference than as a temporary work note

Promotion does not mean the note is final forever. It means the note deserves durable placement and controlled updates.
