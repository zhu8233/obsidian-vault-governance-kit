# Audit Checklist

## Registry checks

- every important topic has a `topic_id`
- important objects appear in registry
- promotion queue entries still point to valid files

## Content checks

- canonical notes still map to intended topics
- working notes are not silently masquerading as canonical
- adapter files still point to `RULES.md`

## Repair classification

- registry patch
- metadata patch
- proposal needed
- Hermes decision needed
