# Intake And Curation Checklist

## Before editing

- Read `RULES.md`
- Resolve `topic_id`
- Resolve target layer
- Check write authority

## During editing

- Keep source references
- Keep raw and curated content distinct
- Avoid canonical phrasing unless the note is actually ready for promotion

## After editing

- update registry if object metadata changed
- append ledger entry for important work
- create promotion proposal if the note now has long-term value
