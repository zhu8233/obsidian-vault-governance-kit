- multi-agent behavior skills
- vault bootstrap templates
- an example governed vault

## Before Publishing

1. Run `python scripts/validate_repo.py`
2. Re-read `README.md`
3. Verify all skill names and prompts look public-ready
4. Confirm no personal absolute paths exist in templates
5. Confirm example vault uses only generic sample data
