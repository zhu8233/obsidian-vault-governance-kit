# Skill Pack

## Purpose

The skill pack teaches agents how to behave inside a governed vault.

It does not replace:

- `RULES.md`
- local adapter files
- registry state

It complements them.

## Skills

### `vault-governance`

Entry skill. Use before reading or editing a governed vault.

### `vault-intake-curation`

Use for raw source handling and curation-layer work.

### `vault-canonical-promotion`

Use when material may become long-lived canonical knowledge.

### `vault-audit-repair`

Use when checking drift, missing metadata, orphan files, or registry mismatches.

### `hermes-coordination`

Use when Hermes acts as coordinator for merge, promotion, or archive decisions.

## Important Limitation

Skills guide behavior. They do not provide hard enforcement. Registry and audit are still required.
